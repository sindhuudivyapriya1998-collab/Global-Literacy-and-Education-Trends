# Global-Literacy-and-Education-Trends
A data analysis project that explores global literacy, illiteracy, education and GDP trend using Python, SQL and Power BI.
The project involves data preprocessing, exploratory data analysis, SQL-based analysis and Power BI visualizations to identify trends, compare regions, and understand the relationship between education and economic development across countries over time.

# Domain
Education Analytics & Socio-Economic Data Analysis

# Objectives
- Analyse literacy and illiteracy trends across countries.
- Compare literacy rates by region.
- Study the relationship between GDP and education.
- Identify countries with high and low literacy rates.
- Built interactive Power BI dashboards for visualization.

# Technologies used
- Pandas
- SQLite
- Power BI
- Google colab
- Github

# Exploratory Data Analysis(EDA)
- Data cleaning

- Summary statistics

- Bivariate analysis

- Average literacy by region
- India's literacy trend
- Top countries with highest illiteracy
- GDP vs schooling analysis

- Male vs female literacy comparison

# Power BI dashboard
The dashboard includes
- Bar charts
- Line chars

- Tables\
These visualization help identify global education trends and compare performance across countries.

# Observation
- Europe has the highest average literacy rate.
- Africa has comparatively lower literacy rates.
- Literacy generally improves with increased years of schooling
- Countries with higher education level often have higher GDP per capita
